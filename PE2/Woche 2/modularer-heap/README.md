# Aufgabe 2: Modularer Heap

1. Implementieren Sie eine Datenstruktur `heap_t` mittels modularer Programmierung. Der Stack soll bei Bedarf automatisch vergrö̈ßert werden, sodass er beliebig viele Werte vom Typ `int` speichern kann. Die Datenstruktur soll dabei die Operationen unterstützen, die in der gegebenen Header-Datei vorgegeben sind. Wenn auf einem leeren Heap ein `minimum` oder `extractMin` ausgeführt wird, soll eine Fehlervariable gesetzt werden.

2. Erstellen Sie einen Testtreiber und Testfälle, um die Korrektheit Ihrer Implementierung nachzuweisen. Test sind grundsätzlich so zu gestalten, dass sie automatisch ablaufen und die Ergebnisse selbsẗ überprüfen.

Informationen darüber, was ein Heap ist, wie man einen Heap mit Hilfe eines Arrays implementiert und wie die einzelnen Operationen implementiert werden können, finden Sie [hier](https://de.wikipedia.org/wiki/Bin%C3%A4rer_Heap).

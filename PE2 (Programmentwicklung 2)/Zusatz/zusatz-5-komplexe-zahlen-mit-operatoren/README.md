# Zusatzaufgabe 5: Komplexe Zahlen mit Operatoren

Erweitern Sie die komplexen Zahlen aus [Zusatzaufgabe 3](https://git.ide3.de/pe2/aufgaben/zusatz-3-komplexe-zahlen).

- Überladen Sie die Operatoren `+`, `-`, `*` und `/` für die komplexen Zahlen. Es soll eine Ausnahme *geworfen* werden, wenn durch 0 dividiert wird. Ändern Sie das Programm `circuit.cpp` so, dass diese Operatorüberladungen für die Berechnungen genutzt werden.

- Die Vergleichsoperatoren `==` und `!=` sollen überladen werden. Überladen Sie außerdem den Ausgabeoperator `<<` sowie den Eingabeoperator `>>` um komplexe Zahlen einfach auf dem Bildschirm ausgeben und von der Tastatur einlesen zu können.

Auch hier gehört das Erstellen von Testtreibern und Testfällen zur Aufgabe,
um die Korrektheit Ihrer Implementierung nachzuweisen.
